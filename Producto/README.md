# CogniMirror - Código Fuente (MVP)

Tecnologías utilizadas: Next.js, React, Supabase, Tailwind CSS, Web Bluetooth API.

## Instrucciones de ejecución local
1. Abrir la terminal en esta carpeta (`Producto`).
2. Instalar las dependencias ejecutando: `npm install`
3. Iniciar el servidor de desarrollo: `npm run dev`
4. Abrir en el navegador: `http://localhost:3000`
